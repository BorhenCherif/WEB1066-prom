# Prometheus

Deploy a prometheus server for CI Analytics on Travis to Heroku.

## Requirements
- This project will require docker
- We will also need a heroku client and account setup with Heroku.com

## Local Setup

```sh
script/build.sh
script/server_local.sh
```
